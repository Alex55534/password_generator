\# 🔐 Password Generator



A simple command-line password generator written in Python. It allows you to create secure passwords with customizable character sets.



\## 🚀 Features



\- \*\*All Characters Mode\*\*: Generates a password using lowercase letters, uppercase letters, digits, and punctuation.

\- \*\*Custom Mode\*\*: Choose from predefined combinations:

&#x20; 1. Letters and numbers

&#x20; 2. Numbers and special characters

&#x20; 3. Special characters and letters

&#x20; 4. Numbers only

&#x20; 5. Special characters only

&#x20; 6. Letters only

\- User‑friendly prompts and input validation.



\## 📦 Requirements



\- Python 3.6 or higher

\- No external libraries needed (uses only built-in modules: `random`, `string`)



\## 🛠️ How to Run



1\. Clone this repository or download the script.

2\. Open a terminal in the project folder.

3\. Run the script:

&#x20;  ```bash

&#x20;  python password\_generator.py
